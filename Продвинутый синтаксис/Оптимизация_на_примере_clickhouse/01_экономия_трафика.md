# Экономия трафика

Сравним, на примере clockhouse, потребление ресурсов при вызове всех полей / только необходимых.

Сначала выполним запрос возвращающий все поля таблицы:
```sql
SELECT *
  FROM orders_good;
```

Затем выполним запрос с учетом принципа экономии трафика. В результирующий набор не должно быть включено кроме того, что неоьходимо для выполнения задачи. Для наглядности и сравнения вернем лишь один столбец.

```sql
SELECT OrderID 
  FROM orders_good;
```

Сравним результаты:

|Тип запроса     |Строк прочитано |Строк в выводе |Байт в выводе |Исп. памяти |Время выполнения |
|----------------|----------------|---------------|--------------|------------|-----------------|
|Все поля (*)    |12_963          |12_963         |5_808_313     |24_937_512  |217              |
|Экономия трафика|12_063          |12_963         |259_525       |3_937_177   |33               |  

Результат очевиден, лишние поля лучше не использовать в выводе.
